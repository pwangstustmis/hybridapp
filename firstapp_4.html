語法、變數、資料類型

Dart為Google開發前端物件導向語言，程式語法類似Java，變數大小寫不同，資料類型包含num(整數或小數)、String(字串)、bool(布林值)、List(陣列)、Set(集合)、Map(對應)。

  • String(字串)：可包含於單引號或雙引號，具變數解析功能，所謂變數解析是指引號內可包含變數名稱，程式執行時會將變數值解析至字串中。 
    • 
int a=1;
double b=1.1;
num c=3;
num d=1.1;
  • num(整數或小數)：包含整數(int)與小數(double)。 
    • 
int a=1;
double b=1.1;
num c=3;
num d=1.1;
  • bool(布林)：包含true、false。 
    • 
bool a=true;
  • list(陣列) 
    • 
var list1=[1,2,3,4];
var list2=['a','b','c','d'];
  • list(陣列)，下列程式定義整數陣列 list1 並利用迴圈印出 list1 每個值。 
    • 
void main() {
  var list1=[1,2,3,4];
  list1.forEach((data){
    print(data);
  });
}
  • set(集合)，集合不像陣列有順序，同時資料包含於 { }中，而且集合元素不限定相同資料類型。 
    • 
void main() {
  var set1={1,2,'abc','def'};
  set1.forEach((data){
    print(data);
  });
}
  • Map(對應)，對應資料類型為一物件，他定義鍵(key)與值(value)對應。 
    • 
void main() {
  var map1={'name':'甲','phone':'1234567'};
  print(map1['name']);
  var map2=new Map();
  map2['id']=1;
  map2['address']='南台街1號';
  print(map2);
}
  • 函數，函數定義包含傳回類型、函數名稱、括符、與代入參數。 
    • 
int mysum(int i,int j) {
  return i+j;
}
void main() {
  print(mysum(10,20));
}
